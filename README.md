# nova-s3

NOVA S3 Storage Service with x402 Payment Integration

## Overview

nova-s3 provides decentralized file storage and retrieval with SKALE and x402 payment. It includes:
- **Server**: FastAPI/Hono-based backend for file upload, download, payment, and expiration management.
- **Client**: Node.js client for easy file operations and payment handling.

## Quick Start

- **Server API Documentation**: [server/API.md](server/API.md)
- **Client Usage Guide**: [client/QUICKSTART.md](client/QUICKSTART.md)

## Features
- Upload, download, list, delete files
- File expiration and renewal
- x402 payment integration
- Docker support

## Environment
- Node.js 20+
- SKALE chain
- AWS S3
